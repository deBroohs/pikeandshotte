# Pike & Shotte Field Companion

Локальное статическое приложение для настольной игры `Pike & Shotte`.

Что внутри:

- трекер полного хода и фаз `Command / Shooting / Hand-to-Hand`
- учёт армий, battalia и статусов unit
- быстрые таблицы по приказам, движению, стрельбе и break tests
- справочник по ключевым правилам core rulebook
- архетипы юнитов и библиотека спецправил
- сценарный prep-блок и цели сторон

Как открыть:

1. Откройте файл `index.html` в браузере.
2. Для быстрого старта нажмите `Демо-состав`.
3. Состояние партии сохраняется в `localStorage` браузера.
4. Для переноса между устройствами используйте `Экспорт` и `Импорт`.

Основные файлы:

- `index.html` — структура приложения
- `styles.css` — визуальная тема
- `rules-data.js` — база правил, архетипов и спецправил
- `app.js` — логика интерфейса и трекера
- `plugins/visitor-counter/runtime.js` — изолированный runtime-плагин для подсчёта посетителей
- `tools/extract-pdf-text.ps1` — локальный утилитарный extractor для текстовых PDF rulebooks

Публичный счётчик посетителей:

1. Откройте `site.config.json`.
2. По умолчанию в правой верхней части сайта на desktop показывается компактная карточка посещений через публичный badge-сервис.
3. В мобильной версии карточка скрыта.
4. Блок `plugins.visitorCounter` оставлен как запасной runtime-вариант и по умолчанию отключён.

Счётчик встроен как отдельный виджет в шапке сайта и не выводится на мобильных экранах.

Источники правил:

- `../Pike & Shotte/Pike & Shotte Rulebook.pdf`
- `../Pike & Shotte/Pike & Shotte - Devils Playground.pdf`

Важно: сценарные ссылки и встроенные PDF-страницы в приложении ожидают,
что каталог `Pike & Shotte` лежит рядом с папкой приложения, уровнем выше.
